# Anti scraping
部分策略：

| 类型           | 说明                                                                                                                                               |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| 自动化工具检测 | 检测 PhantomJS、Selenium、Puppeteer、Playwright 等多种自动化工具，以及自动化工具隐形插件                                                         |
| 隐身模式检测   | 浏览器是否处于匿名模式下                                                                                                                           |
| 代码调试状态检测 | 代码是否处于调试状态                                                                                                                             |
| UA 变量检测    | 用户代理是否有被篡改                                                                                                                               |
| 风险插件检测   | 是否安装存在风险的插件                                                                                                                             |
| Cookie 规范性检测 | Cookie 是否篡改                                                                                                                                |
| 虚拟浏览器检测 | 是否处于仿造的浏览器环境                                                                                                                           |
| 代码篡改检测   | 系统方法是否被 hook                                                                                                                               |
