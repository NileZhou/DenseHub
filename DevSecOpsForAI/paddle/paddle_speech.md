

# Create python venv



```shell

cd <venv dirs>
python3 -m venv paddle
source paddle/bin/activate

# proxy settings

pip install --upgrade pip

# install paddle first
# reference: https://www.paddlepaddle.org.cn/documentation/docs/zh/install/pip/linux-pip.html

# paddle speech
pip install paddlespeech -i https://pypi.tuna.tsinghua.edu.cn/simple

```
